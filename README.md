# JellyReport

JellyReport is a self-hosted reporting system for Jellyfin.

It allows users to report playback issues, missing media problems, subtitle issues, and other problems directly to the server administrator.

## Features

Current:

- Docker deployment
- React frontend
- FastAPI backend
- SQLite configuration storage
- Jellyfin connection setup

Coming soon:

- Media issue reporting
- Discord notifications
- Admin dashboard
- Report history
- Analytics

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/JellyReport.git
cd JellyReport
docker compose up -d

```

Open:
http://YOUR_SERVER_IP:10910
